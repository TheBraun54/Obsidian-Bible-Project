# Lexicon Entry H5855

**Lemma**: עַטְרוֹת שׁוֹפָן

**Transliteration (xlit)**: ʻAṭrôwth Shôwphân

**Pronunciation**: at-roth' sho-fawn'

**Description**:
from the same as עֲטָרוֹת and a name otherwise unused (being from the same as שָׂפַן) meaning hidden; crowns of Shophan; Atroth-Shophan, a place in Palestine; Atroth, Shophan (as if two places).
