# Lexicon Entry H6966

**Lemma**: קוּם

**Transliteration (xlit)**: qûwm

**Pronunciation**: koom

**Description**:
(Aramaic) corresponding to קוּם; {to rise (in various applications, literal, figurative, intensive and causative)}; appoint, establish, make, raise up self, (a-) rise (up), (make to) stand, set (up).
